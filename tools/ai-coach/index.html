Archivos .gpx de tu reloj, Strava o cualquier app de running

Desde Strava

  1. Entra a tu actividad en Strava (web o app)
  2. Haz clic en los 3 puntos (...) y selecciona "Exportar GPX"
  3. Descarga el archivo y súbelo aquí

Desde Garmin / Coros / Suunto / Polar

  1. Conecta tu reloj al ordenador vía USB
  2. Busca la carpeta "GARMIN" / "ACTIVITY" y copia el archivo .gpx o .fit
  3. Si es .fit, conviértelo a GPX usando GPS Visualizer

Desde tu celular (sin reloj)

  1. Usa apps como Strava, Nike Run Club, Adidas Running o Pumatrac
  2. Después de correr, busca la opción "Exportar" o "Compartir como GPX"
  3. Algunas apps permiten exportar directamente; otras requieren usar la web
  4. Si no encuentras la opción, puedes llenar el formulario manual con tus datos aproximados

No importa si tus datos no son perfectos. El análisis se adapta a la información disponible. Un GPX completo da el mejor resultado, pero el formulario manual también entrega observaciones valiosas.
