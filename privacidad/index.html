Privacidad

Política de Privacidad y Datos

Transparencia total. Tus datos de rendimiento te pertenecen a ti, no a nosotros.

Última actualización: 03/07/2026

1. Almacenamiento Local (LocalStorage)

En JPZCH Running Labs no utilizamos bases de datos externas ni servidores para almacenar tus métricas de rendimiento. Toda la información que ingresas —estatura, cadencia, ritmo, y preferencias de entrenamiento— se guarda exclusivamente en tu navegador mediante la tecnología LocalStorage.

Esto significa que tus datos nunca abandonan tu dispositivo. Nosotros, como plataforma, no tenemos acceso a ellos, no los procesamos en ningún servidor, y no los compartimos con nadie. Si cambias de navegador o borras los datos locales, tu perfil se eliminará contigo.

2. Sin Rastreadores Invasivos

No utilizamos cookies de rastreo publicitario, píxeles de seguimiento, ni herramientas de terceros que perfilen tu comportamiento con fines comerciales. Esta plataforma es de código abierto y sostenible por sus propios medios.

Las únicas solicitudes externas que realiza la plataforma son:

  • CDN de Tailwind CSS para los estilos (cargado desde cdn.tailwindcss.com).
  • Envío de formularios a Formspree si decides contactarnos.

3. Formularios de Contacto

Las secciones de Reportar un Error y Sugerir una Herramienta utilizan Formspree, un servicio externo de procesamiento de formularios. Cuando envías un mensaje, este se convierte en un correo electrónico que recibimos directamente.

El correo electrónico que proporcionas (si decides hacerlo) se utiliza exclusivamente para poder responderte o notificarte si tu sugerencia se convierte en realidad. No se agrega a ninguna lista de correo, no se comparte con terceros, y no se usa para marketing. Puedes estar tranquilo.

4. Código Abierto — Transparencia Total

Creemos que la confianza se demuestra, no se declara. Todo el código fuente de JPZCH Running Labs está disponible públicamente en GitHub.

Si tienes conocimientos técnicos, puedes revisar exactamente qué hace la plataforma, cómo procesa los datos, y verificar que no hay procesos ocultos, ni recolectores de información, ni «magia» detrás de las cortinas. No tienes que confiar en nuestra palabra; puedes comprobarlo tú mismo.
